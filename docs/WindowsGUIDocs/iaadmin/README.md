# iaadmin Application

## Overview

iaadmingui (ImgArchive Admin GUI) is a Windows Forms administrative interface application for ImgArchive, a Digital Asset Management (DAM) system designed specifically for photographers to organize, archive, and manage digital images with version control capabilities.

## Purpose and Functionality
iaadmin serves as the administrative configuration and management console for ImgArchive. It provides a graphical interface that:

1.	Configures the ImgArchive system settings
2.	Manages archive operations and metadata
3.	Orchestrates backend command-line tools (primarily iaadmin.exe)
4.	Presents user-friendly forms for complex configuration tasks
The application acts as a frontend wrapper that communicates with the underlying ImgArchive backend tooling through XML-based command execution.
