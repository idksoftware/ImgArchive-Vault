# Configuration Reference

ImgArchive has a flexible method of installation in order to maximize the available hardware. In order to realise this flexibility ImgArchive has a range of installation configuration options. This chapter details these options and how best to use them.

Configuring the installation – takes the standard installation and details how to take this installation and maximize it to fit your requirements.

## The ImgArchive configuration

The ImgArchive configuration is the definition of the location and options used the define the operation of ImgArchive.

This falls into a number of parts:

- Resource locations.
- Working environment locations
- Working environment options

### Resource locations  

The ImgArchive user a number of resources, this is mainly hard drive space in the form of repositories, workspace and catalogues.

This will also include the required backups of the repositories.

### Working environment locations

The Working environment locations were the working environment is located from which the ImgArchive can operate, for example were the log files are located.
