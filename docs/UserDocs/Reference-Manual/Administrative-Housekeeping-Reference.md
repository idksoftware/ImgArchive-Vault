# Administrative Housekeeping Reference
***

This is the command line reference of the administrative house housekeeping tool **iaadmin** “iaadmin.exe”. 

## Name

isadmin <sub-command> <options>…

## Synopsis:

isadmin [-q | –quiet] | [ init <options> …] | [ config <options> …] | [ show <options> …] |

[ archive <options> …] | [ sync <options> …] | [ purge <options> …] | [ validate <options> …] |

[ allow <options> …] | [ about <options> …] 

The list below details the sub commands **iaadmin** will except as a second argument.

1. [init](./INIT-Initialises-the-ImgArchive-environment) – Create ImgArchive working environment
2. [config](./CONFIG-Configures-the-ImgArchive-environment.md) – Configure ImgArchive parameters
3. [setenv ](./SETENV-Sets-environment-variables.md)– Configure ImgArchive parameters– Sets ImgArchive environment variables
4. [show](./SHOW-Shows-the-configuration-ImgArchive.md) – Show how the system is configured and it's status.
5. [archive](./ARCHIVE-Archives-the-repositories-to-long-term-storage.md) – These options control the archive activities.
6. [backup](./BACKUP-Cold-Backup-the-repositories-to-long-term-storage.md) – These options control the backup activities.
7. [sync ](./SYNC-Synchronises-the-repositories-with-the-associated-backups.md)– This command synchronises the primary archives with there associated backups.
8. [purge](./PURGE-Purges-deleted-images-from-archive.md) – Purges deleted images from archive
9. [validate ](./VALIDATE-Validates-the-archive-for-correctness.md)– Validate commands
10. [allow](./ALLOW-Which-image-file-extensions-into-the-archive.md) – Controls which image file extensions are allowed into the archive.
11. [naming](./NAMING-File-Naming-Convention.md) - Controls the renaming of images in the archive.
12. [about](./ABOUT-Administrative-configuration-tool.md) – prints the version information
13. [help ](./HELP-On-Administrative-Housekeeping-commands.md)- On Administrative Housekeeping commands
