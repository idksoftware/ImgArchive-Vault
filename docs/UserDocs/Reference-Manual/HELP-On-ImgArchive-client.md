# HELP - On ImgArchive client.

# Name

iaarc help <options>

## Synopsis

iaarc help [–out=<options>] [–file=<filename>]

## Description

The command prints the version information.

## Options

–out :

Output type: text, xml, json or html.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
