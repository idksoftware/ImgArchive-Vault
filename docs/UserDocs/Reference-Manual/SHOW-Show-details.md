# SHOW – Show details

# Name

iaarc show <sub-category> <sub-category-options> <options>

This command allows you to  show or list images that match images with criteria of interest.

## Synopsis

iaarc show <sub-category> [<sub-category-options>] [–out=<options>] [–file=<filename>]

## Description

The command prints out details of the archive depending on the sub category.

 

## Options

–out :

Output type: text, xml, json or html.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
