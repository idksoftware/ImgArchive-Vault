# UNDELETE – Undelete images in the archive

# Name

iavault undelete <options>

## Synopsis

iavault undelete [–out=<options>] [–file=<filename>]

## Description

This command undeletes images that have previously been deleted. Its state will be undeleted. undelete impage files can then checked-out as normal.

## Options

–out :

Output type: text, xml, json or html.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
