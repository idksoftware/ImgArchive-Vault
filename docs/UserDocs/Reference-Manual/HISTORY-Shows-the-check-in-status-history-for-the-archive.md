# HISTORY – Shows the check-in status history for the archive

# Name

iaarc history <options>

## Synopsis

iaarc history  [–scope=<options>] [–format-type=<filename>] [--file=path

## Description

While the archive is being used, All the activity that is being  generated is logged into the system history. For example, when a user checks an image  out of the archive this checkout activity is logged in the system history. This gives a running historical log of what happened in the archive when.

The history log can be displayed at any time by using the history command. As the history log is a chronicle of the archive parts of that chronicle can be displayed for example if you wish to display say, the last activities over the last month you can scope the Chronicle to just  a display the last month. To do that you need to use the scope option.

This command also has two other options one is to display the  history log in several different output formats such as XML Jason HTML plus plaintext. The other is to redirect the output from going to the screen to a designated file. 

## Options

–out :

Output type: text, xml, json or html.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
