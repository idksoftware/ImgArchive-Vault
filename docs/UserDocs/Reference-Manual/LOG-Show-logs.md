# LOG – Show logs

# Name

iaarc log <options>

## Synopsis

iaarc log [–out=<options>] [–file=<filename>]

## Description

This command lists the version information for each image 

## Options

–out :

Output type: text, xml, json or html.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
