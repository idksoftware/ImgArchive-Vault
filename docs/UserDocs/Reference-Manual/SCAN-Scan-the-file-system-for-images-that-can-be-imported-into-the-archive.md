# SCAN - Scan the file system for images that can be imported into the archive

# Name

iaarc scan <options>

## Synopsis

iaarc scan [--dups-only] [–out=<options>] [–file=<filename>] [--source-path] [--recursive]

## Description

The command prints the version information.

## Options

--source-path

Source path from which to start the scan.

--recursive

Include recursive sub-folders.

–out :

Output type: text, xml, json or html.

--dups-only:

Find duplicates and put into the output.

## Syntax


out=[plain] | [xml] | [json] | [html]

–file :

output file name. 
Syntax:

file=<filename>
