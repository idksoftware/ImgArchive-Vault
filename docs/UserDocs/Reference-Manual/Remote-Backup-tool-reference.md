# Remote Backup tool reference 

This is the command line reference of the Remote Backup tool **iarbk** “iarbk.exe”.

## Name

iarbk <sub-command> <options>…

## Synopsis:

iarbk <sub-command> <options>…

[ archive <options> …] | [ sync <options> …] | [ purge <options> …] | [ validate <options> …] |

[ allow <options> …] | [ about <options> …]

The list below details the sub commands **iaadmin** will except as a second argument.

1. [init](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/799375468) – Create remote backup working environment
2. [config](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/799080632) – Configure remote backup working parameters
3. [setenv ](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/853114881)– Sets remote backup environment variables
4. [show](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/799080639) – Show how the system is configured.
5. [sync ](./iarbk-sync.md)– This command synchronises the primary archives with there associated remote backups.
6. [validate ](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/799375489)– Validate commands
7. [about](https://idk-solutions.atlassian.net/wiki/spaces/ImgaWiki/pages/799047894) – prints the version information
8. [help ](https://idk-solutions.atlassian.net/wiki/pages/resumedraft.action?draftId=799375404)- On Administrative Housekeeping commands
