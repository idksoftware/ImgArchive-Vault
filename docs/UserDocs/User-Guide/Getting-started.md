As with all software products ImgArchive will need a host computer to install it on. This computer will depend on your requirements. The System Requirements section in this guide will give you guidance on the specification of the machine that will best meet your particular requirements. See [System Requirements](System-Requirements.md).

In this early version, the product consists of several command-line applications. However, this is changing rapidly as a Windows GUI becomes available. Note, there is a Administration and Setup as part of this release.

ImgArchive is downloaded from https://www.imgarchivevault.com/download . The installation section details how to carry out a simple default installation. See[ Installing](../Installing.md).

To make the best use of ImgArchive you may need to configure the default installation. The [Configuring the installation](./Configuring-the-installation.md) section details how to configure the installation to tailor ImgArchive to your requirement.

Finally, before adding images to the archive the default import workflow may need to also be tailored to you requirements. see  [Configuring the import work flow](./Configuring-the-import-work-flow.md).

Hopefully this will be a once only process and once configuring is over, this will not be required for sometime.
